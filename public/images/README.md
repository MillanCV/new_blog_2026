# Images Directory

Place your photography images here.

## File Naming Convention

Use descriptive, sequential filenames for your images:

- `rehearsal-001.jpg`, `rehearsal-002.jpg`, etc.
- `portrait-001.jpg`, `portrait-002.jpg`, etc.
- `personal-001.jpg`, `personal-002.jpg`, etc.

## Image Requirements

- **Format**: JPG or PNG
- **Recommended width**: 1920-2400px for best quality
- **Optimize**: Compress images before uploading to reduce file size
- **Aspect ratio**: Mix of landscapes and portraits works well

## Quick Start

1. Copy your images into this directory
2. Update `src/data/projects.ts` with the correct image paths
3. Run `npm run dev` to see your portfolio

## Example Structure

```
images/
├── rehearsal-001.jpg
├── rehearsal-002.jpg
├── portrait-001.jpg
├── portrait-002.jpg
└── personal-001.jpg
```
